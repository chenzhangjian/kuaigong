## 五、APP功能介绍

### 5.1 APP简介

快工是一款即时工作匹配平台移动应用，通过移动互联网技术连接雇主与工人，实现零工经济的高效对接。平台提供任务发布、在线接单、订单管理、在线支付、评价反馈等功能。

### 5.2 核心功能模块

| 模块 | 功能说明 |
|------|----------|
| **用户系统** | 注册登录、个人中心、实名认证、角色切换 |
| **任务系统** | 任务发布、任务浏览、任务搜索、任务筛选 |
| **订单系统** | 订单管理、状态跟踪、订单详情 |
| **支付系统** | 钱包管理、充值提现、交易记录 |
| **评价系统** | 用户评价、评分展示 |
| **消息系统** | 系统通知、订单消息 |

### 5.3 服务对象

| 角色 | 服务内容 |
|------|----------|
| **雇主** | 发布零工任务、筛选工人、支付结算、评价服务 |
| **工人** | 浏览任务、申请接单、完成工作、获取报酬 |

## 六、APP权限使用说明

### 6.1 权限清单

| 权限名称 | 权限用途 | 是否必需 |
|----------|----------|----------|
| **相机权限** | 拍摄照片上传头像、任务图片 | 可选 |
| **相册权限** | 选择图片上传头像、任务图片 | 可选 |
| **位置权限** | 获取附近任务、导航功能 | 可选 |
| **网络权限** | 数据请求、API调用 | 必需 |
| **存储权限** | 缓存数据、图片存储 | 必需 |

### 6.2 权限使用场景

```
1. 相机权限
   - 用户上传头像时拍照
   - 雇主发布任务时拍照上传任务图片
   - 工人完成任务时拍照确认

2. 相册权限
   - 用户选择本地图片作为头像
   - 雇主选择本地图片发布任务

3. 位置权限
   - 工人查看附近任务
   - 显示任务地点位置
   - 导航到工作地点
```
