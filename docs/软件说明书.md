## 一、软件概述

### 1.1 软件简介

"快工"是一款基于移动互联网的即时工作匹配平台应用软件，旨在连接雇主与工人，实现零工经济的高效对接。该软件采用 Expo + React Native + Express 技术架构开发，支持 Android、iOS 双平台运行。

### 1.2 软件用途

本软件主要用于：

1. **雇主端**：发布零工任务、筛选工人、管理订单、支付结算
2. **工人端**：浏览任务、接单赚钱、订单管理、收益提现

### 1.3 运行环境

- **移动端**：Android 5.0+ / iOS 12.0+
- **服务端**：Node.js 18+ / PostgreSQL 14+
- **网络环境**：需要互联网连接

## 三、操作说明

### 3.1 新用户注册流程

1. 打开"快工"APP，进入登录页面
2. 点击"注册"按钮，切换至注册模式
3. 输入手机号码
4. 点击"获取验证码"，接收并输入短信验证码
5. 设置登录密码（6位以上）
6. 选择用户身份：工人或雇主
7. 勾选同意《用户协议》和《隐私政策》
8. 点击"注册"完成注册

### 3.2 雇主发布任务流程

1. 登录后，点击底部导航"首页"
2. 点击右下角"+"按钮或"发布任务"
3. 填写任务信息：
   - 输入任务标题
   - 填写任务描述
   - 选择任务分类
   - 设置工作地点
   - 设置薪资预算
   - 选择紧急程度
   - 上传相关图片（可选）
4. 点击"发布"完成任务发布

### 3.3 工人接单流程

1. 登录后，浏览首页任务列表
2. 可通过分类、地点、薪资筛选任务
3. 点击任务卡片查看详情
4. 确认任务信息后，点击"申请接单"
5. 可填写申请说明（可选）
6. 等待雇主确认
7. 雇主确认后，订单进入"进行中"状态

### 3.4 订单完成流程

1. 工人完成任务后，通知雇主
2. 雇主验收任务，确认完成
3. 订单状态变更为"已完成"
4. 双方互相评价
5. 系统自动结算款项至工人钱包

### 3.5 提现流程

1. 进入"我的"页面，点击"钱包"
2. 查看账户余额
3. 点击"提现"按钮
4. 输入提现金额
5. 选择提现方式（银行卡/支付宝等）
6. 确认提现信息，提交申请
7. 等待审核到账

## 四、技术特点

### 4.1 跨平台技术

采用 React Native + Expo 框架开发，实现一套代码同时运行于 Android 和 iOS 平台，保证用户体验一致性。

### 4.2 实时通信
