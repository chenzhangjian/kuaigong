# 软件著作权申请指南

## 一、申请所需材料

### 1. 申请表
- 计算机软件著作权登记申请表（在线填写并打印）

### 2. 身份证明文件
- **个人申请**：身份证复印件
- **企业申请**：企业法人营业执照副本复印件（加盖公章）

### 3. 软件鉴别材料

#### 3.1 软件说明书
- **页数要求**：不少于10页
- **内容要求**：包含软件功能说明、操作流程、界面截图等
- **格式要求**：A4纸打印，左侧装订

#### 3.2 源代码文档
- **页数要求**：前30页 + 后30页（共60页）
- **行数要求**：每页不少于50行
- **格式要求**：A4纸打印，去除空行，左侧装订
- **注**：如代码总页数不足60页，应提交全部源代码

### 4. 其他材料
- 软件归属权证明（如委托开发合同、合作协议等）
- 继承、受让软件著作权的证明文件（如有）

---

## 二、申请流程

### 步骤一：准备材料
1. 完成软件开发和测试
2. 编写软件说明书
3. 整理源代码文档

### 步骤二：在线填报
1. 登录中国版权保护中心官网
2. 注册/登录账号
3. 在线填写软件著作权登记申请表
4. 上传相关材料

### 步骤三：提交材料
1. 打印申请表（签字/盖章）
2. 准备纸质材料
3. 邮寄或现场提交至中国版权保护中心

### 步骤四：审核登记
1. 形式审查（1-2个月）
2. 实质审查
3. 颁发证书

---

## 三、材料准备要点

### 软件说明书要点

1. **软件基本信息**
   - 软件名称、版本号
   - 开发完成日期
   - 首次发表日期

2. **软件功能介绍**
   - 功能模块说明
   - 技术特点
   - 运行环境

3. **操作流程说明**
   - 安装/部署流程
   - 主要功能操作步骤
   - 界面截图配合说明

4. **注意事项**
   - 使用截图需真实有效
   - 界面截图应标注功能说明
   - 文字描述清晰准确

### 源代码文档要点

1. **代码格式**
   - 使用等宽字体（如 Courier New）
   - 字号建议：小四号（12pt）
   - 页边距：上下左右各2cm

2. **代码选择**
   - 优先选择核心功能代码
   - 包含用户认证、数据处理等关键逻辑
   - 代码应完整、可编译

3. **代码排版**
   - 去除多余空行
   - 保持代码缩进格式
   - 添加必要注释

## 四、时间与费用

### 审查周期
- **普通程序**：30-60个工作日
- **加急程序**：3-5/10-15/20-30个工作日可选（需额外费用）

### 官方费用
- **软件著作权登记费**：免费（自2017年4月1日起免征）
- **加急费用**：根据加急天数收取

## 六、注意事项

1. **软件名称规范**
   - 格式：品牌 + 功能描述 + 软件/系统/平台
   - 示例：快工即时工作匹配平台软件
   - 版本号：V1.0

2. **开发完成日期**
   - 应为软件实际开发完成的日期
   - 早于首次发表日期

3. **首次发表日期**
   - 可与开发完成日期相同
   - 或填写实际发表/上线日期

4. **权利取得方式**
   - 原始取得：自行开发
   - 继受取得：转让、继承等

5. **权利范围**
   - 全部权利：财产权 + 人身权
   - 部分权利：需明确具体权利项
