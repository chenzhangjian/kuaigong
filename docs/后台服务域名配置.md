# 快工APP 后台服务域名配置

## 一、服务域名清单

### 1. 核心API服务

| 服务名称 | 环境变量 | 说明 |
|----------|----------|------|
| **后端API服务** | `EXPO_PUBLIC_BACKEND_BASE_URL` | 主要业务API接口 |
| **对象存储服务** | `COZE_BUCKET_ENDPOINT_URL` | 文件上传存储（头像、图片等） |

### 2. 当前开发环境配置

| 服务 | 地址 | 端口 |
|------|------|------|
| 后端API | http://localhost | 9091 |
| 前端Web | http://localhost | 5000 |

### 3. 生产环境配置

生产环境域名由系统自动注入，通过环境变量 `EXPO_PUBLIC_BACKEND_BASE_URL` 获取。

## 二、API接口清单

### 基础路径

所有API均以 `/api/v1` 为前缀

```
${EXPO_PUBLIC_BACKEND_BASE_URL}/api/v1/*
```

### 接口模块

| 模块 | 路径前缀 | 功能说明 |
|------|----------|----------|
| 认证 | `/api/v1/auth` | 登录、注册、验证码、密码重置 |
| 任务 | `/api/v1/tasks` | 任务发布、查询、申请 |
| 订单 | `/api/v1/orders` | 订单管理、状态更新 |
| 支付 | `/api/v1/payments` | 钱包、充值、提现、交易记录 |
| 位置 | `/api/v1/location` | 地理位置服务 |
| 认证审核 | `/api/v1/verification` | 实名认证审核 |

## 三、对象存储服务

### 配置参数

| 参数 | 环境变量 | 说明 |
|------|----------|------|
| Endpoint | `COZE_BUCKET_ENDPOINT_URL` | 存储服务端点 |
| Bucket | `COZE_BUCKET_NAME` | 存储桶名称 |
| Region | cn-beijing | 服务区域 |

### 使用场景

- 用户头像上传
- 任务图片上传
- 认证材料上传

### 文件路径规范

```
avatars/{userId}_{timestamp}.{ext}     # 用户头像
tasks/{taskId}_{index}_{timestamp}.{ext}  # 任务图片
verification/{userId}_{type}.{ext}     # 认证材料
```

## 四、环境变量配置

### 必需环境变量

| 变量名 | 必需 | 说明 |
|--------|------|------|
| `EXPO_PUBLIC_BACKEND_BASE_URL` | ✅ | 后端API地址 |
| `COZE_BUCKET_ENDPOINT_URL` | ✅ | 对象存储端点 |
| `COZE_BUCKET_NAME` | ✅ | 对象存储桶名 |
| `JWT_SECRET` | ✅ | JWT签名密钥 |
| `DATABASE_URL` | ✅ | 数据库连接字符串 |

### 可选环境变量

| 变量名 | 默认值 | 说明 |
|--------|--------|------|
| `PORT` | 9091 | 服务端口 |
| `NODE_ENV` | development | 运行环境 |

## 五、前端配置使用

### API调用示例

```typescript
// client/utils/api.ts
const BASE_URL = process.env.EXPO_PUBLIC_BACKEND_BASE_URL || 'http://localhost:9091';

// 所有API请求
const response = await fetch(`${BASE_URL}/api/v1/auth/login`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ phone, password }),
});
```

### 获取环境变量

```typescript
// 前端直接使用
const apiUrl = process.env.EXPO_PUBLIC_BACKEND_BASE_URL;

// 或通过api工具类
import api from '@/utils/api';
const result = await api.login(phone, password);
```

## 六、域名备案要求

### 需要备案的域名

| 域名类型 | 是否需要备案 | 说明 |
|----------|--------------|------|
| API域名 | ✅ 需要 | 后端API服务域名 |
| 对象存储域名 | ⚠️ 可能需要 | 如使用自定义域名需备案 |
| CDN域名 | ⚠️ 可能需要 | 如使用CDN加速需备案 |

### 备案主体

所有域名需以 **徐州市快工科技有限公司** 名义备案

## 七、安全配置

### HTTPS要求

- 生产环境必须使用HTTPS
- SSL证书需为有效证书
- 证书需覆盖所有服务域名

### 跨域配置

后端已配置CORS：

```typescript
// server/src/index.ts
app.use(cors());
```

### 安全建议

1. 生产环境禁用 `*` 跨域
2. 配置白名单域名
3. 启用请求频率限制
4. 启用SQL注入防护
